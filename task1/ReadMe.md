# Function Plotter

## programming language
c++ / C programming languages

## Flow 
1. enter your desired function of x  
2. specifiy the min value and max value for variable x
3. click Draw button
4. you can use "zoom" buttons for better view

## Tools used
- Qt creator
- Doxygen 

## External Libraries
